# MiniShop多商铺SAAS系统API后台
>根据MINIshop单店铺系统框架作为基础，搭建了这一套API后台系统，后台包含了电子商务所需要的基本功能，高级功能均可以在这套结构之上进行快速二开。
本框架将长期维护。
> >本系统框架选用Laravel 6 LTS版本，遵循RESTful API设计，用户认证采用JWT无状态的Bearer认证方法
